# Git How-to

## Working with branches

List all local branches
```
git branch
```

List local+remote branches
```
git branch -a
```

When working on a branch and want to update it with the latest from main
```
git checkout main
git pull
git checkount my_branch
git merge main
```

Delete a branch
```
git branch -d my_branch
```
